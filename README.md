# MATH-UoL-Hackathon

## Data

This project uses the **CMS DE-SynPUF Prescription Drug Events (PDE), Sample 1 (2008–2010)**.

This project has been buildt with using fully synthetic data designed for training and software development — no real patient data is used.

**Download instructions:**
1. Download the ZIP directly: https://downloads.cms.gov/files/DE1_0_2008_to_2010_Prescription_Drug_Events_Sample_1.zip
2. Unzip and place the `.csv` file in a `/data` folder at the root of this repo.
3. The `/data` folder is listed in `.gitignore` and will not be committed.

Codebook (column definitions): https://www.cms.gov/files/document/de-10-codebook.pdf-0
